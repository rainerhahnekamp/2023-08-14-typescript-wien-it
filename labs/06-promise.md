# TypeScript Workshop - Promise

1. Study the example shown by the trainer
2. Add `loadByName` to the `AbstractPassengerManager`. It should load passengers via HTTP. The url is `http://www.angular.at/api/passenger?name=Muster`.
3. Use that method in your application and show the results
4. Test your implementation
5. Create a method `observableByName`. It should work like the `loadByName` but should return an `Observable` instead of a `Promise`.
6. Test your implementation
